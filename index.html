チーム作りの基礎知識が学べる​

ビジネスもスポーツも、成功のカギは基礎にあります。
多くのリーダーは、自己の経験と独学を頼りに試行錯誤しながらマネジメントを行っています。その結果、リーダーごとにチームのパフォーマンスに大きな差が生じてしまいます。

01

Learn the basic knowledge of team building

高度なチーム作りが求められる
プロスポーツの実例が学べる​

不確実性の高いスポーツチームにおけるチーム作りの実例を基に、基礎知識を掛け合わせることで、納得感と説得力を兼ね備えた、理解しやすく実践に役立つ情報をお届けします。

02

Study real-life examples of advanced team building in professional sports

圧倒的な対話量による探求学習​

Service

人材育成プログラム

( BuilDoor for リーダーズ​ )

リーダーがビルドアップの方針を作る

確かな方針と適切な環境が整えば、メンバーは自発的に行動を起こします。このプログラムでは、プロスポーツ選手やチーム監督との双方向の対話を通じて、リーダーとしての素養を磨いていきます。

異業種リーダー​交流型

異業種リーダーの交流により、新たな気づきを学ぶと共に、ビジネス拡大を目的としたコミュニティ型のプログラム​

社内リーダー育成・交流型​

人材育成プログラム

( BuilDoor for アクティブチーム​​ )

ビルドアップを実行するのはチームメンバー

チーム内のコミュニケーションの質を高めることが重要です。そのため、プロスポーツチームが切磋琢磨するクラブハウスやキャンプ地など、非日常的な環境でプログラムを実施します。この環境の中で学び、各メンバーの成長を促進します。

新卒採用マッチングプログラム

( BuilDoor for Newフレッシャーズ​​​ )

チームビルディング力に長けている 「体育会系学生新卒」と企業のマッチングを実現

リアルコミニュケーションが減っている大学環境ではありますが、「体育会学生」は勝利のために日々チームメンバーと切磋琢磨しています。企業の将来のリーダー候補になり得る学生とのマッチングを提供してきます。

チームビルディング指導者​

福富信也氏

横浜F・マリノスコーチを経て、現在は東京電機大学理工学部の教員。 Jリーグ監督に必要なS級ライセンス講習会の講師を務める。

2016-17年北海道コンサドーレ札幌（J２優勝、J１昇格）、2018-19年ヴィッセル神戸（天皇杯優勝）、2020-21年ラグビーNTTdocomo（リーグワン参入）、2022-23藤枝MYFC(J2初昇格）、2024年からはFC東京のアドバイザーに就任。

スポーツチームでの豊富なチームビルディングの実績をもとに、企業向けの研修も行っており、大企業から中小企業まで幅広い層に支持されています。著書も多数あり、特に『「脱 トップダウン思考」（東京法令出版）※2019』は高い評価を受けています。
